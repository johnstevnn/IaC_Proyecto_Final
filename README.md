### Comandos para Ejecurtar Terraform
##### Clonar Repositorio:  
``` 
git clone https://github.com/johnstevnn/IaC_Proyecto_Final.git
```
##### Inicializar Terraform: 
``` 
terraform init
```

##### Verificar los recursos a crear: 
```
terraform plan
```
##### Crear la infraestructura:
```
terraform apply
```
